# RAWG

An app for viewing games of different genres and more detailed information about a particular game.
The application works based on the [RAWG API](https://rawg.io)

- Kotlin

- Coroutines + Flow
- Retrofit + OkHttp
- Hilt
- Room
- Custom Navigation
- Coil
- Custom Paging
